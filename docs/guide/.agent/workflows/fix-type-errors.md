# Command: /fix-type-errors

## 目的 (Purpose)
「とりあえず動く」状態から「絶対に壊れない」状態（Type-Safe）へ昇華させるため、TSの厳密な型定義を強制適用します。

## 実行時に自動で行われる推論とアクション (Actions)
1. バックグラウンドで `npx tsc --noEmit` を実行し、全型エラーをリストアップする。
2. エラーメッセージから原因になっているファイルを特定し、`any` や未定義の型 (`unknown`) を使っている箇所を特定する。
3. Zod等のバリデーションライブラリに基づいて、Interface/Type Definitionを自動生成・補完する。
4. 対象ファイルを上書きし、再度 `npx tsc --noEmit` が通ることを検証し、完了メッセージを表示する。
