# 管理画面 要件定義書 v0.1

---

## 概要

AI情報集積システムの運用を支援する管理画面の要件定義。
バッチ処理の監視・操作、コンテンツの品質管理、ソース管理を主な用途とする。

---

## 画面一覧

### 1. ダッシュボード（トップ）

システム全体の状態を一目で把握する画面。

表示項目
- 各バッチの最終実行日時・ステータス（成功 / 失敗 / 実行中）
- 本日の取込記事数・公開記事数・取下記事数
- フェッチャーごとの取得件数サマリー
- 要対応件数バッジ（スコア境界値・タグ承認待ち・エラー）

---

### 2. バッチ管理

#### 2-1. バッチ状況確認

- 日次A / 日次B / 週次 / 月次 / 逐次 それぞれの実行履歴一覧
- 直近10件の実行ログ（開始時刻・終了時刻・処理件数・ステータス）
- エラー発生時は詳細ログを展開表示

#### 2-2. バッチ手動トリガー

- 各バッチを即時実行するボタン
- 実行中は二重起動を防ぐためボタンをロック
- 実行結果をリアルタイムで画面に表示

#### 2-3. エラーログ閲覧

- バッチ・フェッチャー別のエラー一覧
- エラー内容・発生日時・対象URL を表示
- 解決済みマークをつけて管理できる

---

### 3. 記事管理

#### 3-1. 記事一覧

- 絞り込み条件: カテゴリ / ソース / is_published / スコア範囲 / 日付範囲
- 一覧表示項目: タイトル・ソース・カテゴリ・スコア・公開状態・取込日時
- スコア 0.6〜0.8 の記事はハイライト表示（レビュー推奨）

#### 3-2. 記事編集

- タイトル・サマリーの手動編集
- タグの追加・削除
- カテゴリの手動変更
- スコアの手動上書き（理由記入必須）

#### 3-3. 記事操作

- 強制公開（スコア不足でもL4に即時反映）
- 取下（is_published=false に戻す・サイトから即時非表示）
- 重複記事のマージ（同一ニュースが複数ソースから来た場合）

---

### 4. タグ管理

#### 4-1. タグマスター一覧

- 登録済みタグ一覧（tag_name / article_count / confidence_score / last_seen）
- ステータス: 正式登録済み / 承認待ち / 却下済み

#### 4-2. タグ承認ワークフロー

二段階ゲートの人間判断部分

```
候補タグが表示される
  ↓
① article_count が閾値（初期値: 10件）以上か確認
② Google Trends スコアを参照表示（手動確認）
  ↓
承認 → tags_masterに正式登録
却下 → 候補から除外（一定期間再候補にしない）
```

#### 4-3. タグ操作

- タグのマージ（例: `llm` と `large-language-model` を統合）
- タグの手動追加（マスターに直接登録）
- タグの削除（関連記事からも除去するか選択）

---

### 5. ソース管理

#### 5-1. フェッチャー一覧

- 登録済みソース一覧（name / category / 取得方式 / 最終取得日時）
- ON/OFF トグル（即時反映）

#### 5-2. フェッチャー状態確認

- ソースごとの直近取得件数推移
- エラー率・連続失敗回数の表示
- 連続失敗が一定回数を超えたら自動でOFFにして通知する（推奨）

---

### 6. 逐次操作（緊急対応）

管理画面から即時実行できる緊急操作。

- 特定記事の強制取下（炎上・誤情報対応）
- 特定ソースの即時停止
- タグマスターの緊急クリア
- 日次Bバッチ（Webサイト反映）の即時手動実行

## 権限設計（暫定）

| 操作 | 管理者 | オペレーター |
|------|--------|------------|
| バッチ手動トリガー | ✅ | ✅ |
| 記事編集・取下 | ✅ | ✅ |
| 強制公開 | ✅ | ✅ |
| タグ承認・削除 | ✅ | ✅ |
| フェッチャーON/OFF | ✅ | ✅ |
| タグマスター緊急クリア | ✅ | ❌ |
| ソース削除 | ✅ | ❌ |

---

## 優先度

### すぐ必要（運用開始前に必須）
- ダッシュボード
- バッチ状況確認・エラーログ
- 記事一覧・取下・強制公開
- フェッチャーON/OFF

### 運用開始後に追加
- タグ承認ワークフロー
- 記事マージ
- フェッチャー状態の推移グラフ

### 後回しでよい
- 権限管理の細分化
- L3アクセス情報の可視化ダッシュボード
- 月次統計レポート
