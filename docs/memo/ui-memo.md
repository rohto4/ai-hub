# 決めごと（取りやめない限り勝手に変えないこと）
1. サイドバーは中央の横幅を確保するために左側だけにする
2. 左サイドバーのNAVIGATE・タイトル・説明は全て要らない。太文字で堂々と「カテゴリ」とだけ置く。
3. カテゴリ別の説明はいらない
4. 100文字/200文字の切替はヘッダにつける。要約文 短 長とする。
5. ランダム(文字色濃)／新着順(文字色薄)／ユニーク順(文字色薄)は各エリアのヘッダに表示して、どれをクリックしてもそこへ移動できるように。
6. 24時間、1週間、1か月はday/week/monthにしてヘッダに載せる。中央上部からは消す。
7. フォントは等幅フォントで文字数と横幅を管理しやすくする。


# 要望（一旦適用すること）
1. 角が丸すぎると情報量が減るため、基本的には丸くする領域をいまの1/4程度にする
2. 左サイドバーとヘッダは基本的に丸くしない
3. 左サイドバーの「FOCUS TOPIC」を「注目タグ」に変更。
4. 左サイドバーの注目タグを複数選択可能にする
5. 左サイドバーの注目タグは一週間以内の記事に多いタグ10個をピックアップして表示する
6. 左サイドバーのカテゴリの左の添え色は良い感じ。もう少しカラー部分を太く、濃くする。
7. 左サイドバーのカテゴリの背景色は白ではなく透明に。フォーカスされた時だけ左の添え色より一段階薄い色がフワっと浮かび上がるように。
8. 中央上部のFOCUS、10件表示中です。は要らない。
9. 中央上部の下矢印ランダム表示、下矢印新着順、下矢印ユニーク順というのは無くす
10. 中央上部のタグフィルタは消す。
11. OGP共有ボードの黒いリンクを消す
12. 要約文 長 モードにしたら記事カードも大きくしてください。
13. 検索画面のアニメーションは良い感じ。デフォルトで文字入力エリアは白い方がいい
