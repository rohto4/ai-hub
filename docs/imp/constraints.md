# AI Trend Hub 制約・非実装一覧

最終更新: 2026-03-06

---

## 1. 実装しないもの（スコープ外）

### データ収集・解析
- **ジャンル横断 Topic Group の自動統合** — 同ジャンル内コサイン類似度 0.8 以上のみ統合。動画/ブログ/公式の束ねは行わない
- **低信頼クラスタのレビューキュー** — 0.8 未満は別表示のみ。人手レビューキューは設けない
- **ユーザーによる検索ワード入力** — MVP では収集ソースは運営管理のみ。上級者向け入力は後続フェーズ

### 通知・配信
- **メール通知** — Web Push のみ。メール配信は実装しない
- **ユーザーによる通知時刻カスタマイズ** — 初期は 07:00 / 12:00 / 18:00 固定。変更 UI は後続フェーズ

### UI・表示
- **スコア内訳のユーザー向け公開** — rank_scores の内訳は管理画面のみ。一般ユーザーには非表示
- **モバイルでの分析・管理画面** — 分析・管理は PC 専用。モバイルは閲覧特化

### インフラ・更新方式
- **5 秒ポーリング** — サーバー負荷上の理由で採用しない。フォーカス時即時取得 + 5 分バックグラウンドポーリングに留める
- **Supabase Realtime / SSE（P0 段階）** — 負荷検証後に P2 で移行検討。MVP では使わない

---

## 2. 禁止事項・制約

### 法務・著作権
- **本文キャッシュ 7 日超の保存禁止** — 本文テキストは取得後 7 日で削除。メタデータ・要約は永続可
- **引用時のソースリンク省略禁止** — 要約・サムネイル含め、常にオリジナル記事 URL を明示する

### データアクセス
- **RLS バイパス禁止** — 公開読取 / 内部書込の権限境界を越えるクエリは実装しない
- **ジャンル境界を越えた重複統合禁止** — hash 重複判定・Topic Group 統合ともに同ジャンル内に限定

### AI 生成品質
- **100 文字要約の超過・過少を黙認しない** — バリデーション違反は再生成または棄却。違反率 1% 未満を維持
- **禁止語を含む要約の公開禁止** — 禁止語バリデーター通過前の公開は行わない

### 運用
- **ユーザーに見えない形でのスコア操作禁止** — ランキングロジックの変更は管理画面ログに記録する
